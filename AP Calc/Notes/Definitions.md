The derivative of a function
$$\begin{align}
\frac{d}{dx}f(x)=\lim_{ h \to 0 } \frac{f(x+h)-f(x)}{h}
\end{align}$$

$f$ is Continuous if
$$\begin{align}
\lim_{ x \to a^+ }f(x)=\lim_{ x \to a^- }f(x)=f(a)
\end{align}$$

$f$ is Differentiable if
$$\begin{align}
f'(a)=\lim_{ h \to 0^\pm } \frac{f(a+h)-f(a)}{h}
\end{align}$$
